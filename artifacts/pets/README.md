# Codex Pet Artifacts

Each pet directory keeps the generated Codex-ready package plus QA artifacts.

## Directories

- `kingking/` — crowned white companion.
- `booriboorimon/` — 부리부리몬 pig companion.

## Layout

- `install/` — files copied into `~/.codex/pets/<pet-id>/`.
- `final/` — final atlas outputs from the hatch-pet pipeline.
- `qa/` — validation, frame inspection, contact sheet, and animation previews.
- `source/` — original reference image used for generation.

To use a pet in Codex, refresh custom pets in `Settings > Appearance > Pets`.
